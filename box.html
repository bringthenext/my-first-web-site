box model

박스모델 CSS라고 되어있는 부분에 링크를 걸어볼게요.

제목은 화면 전체를 쓴다. 하지만 링크는 딱 필요한 것만 사용한다.


  • 화면 전체를 쓰는 태그 : block level element
  • 자기 자신의 부피만큼 갖는 태그 : inline element

이에 대하여, 제목이 화면 전체를 쓰고 링크는 자신의 크기만큼 사용한다는 것은 기본값일 뿐이다. 스타일태그에서 display inline이나 block을 사용하면 상태를 변경할 수 있다.
border를 중복 입력하게 되는 건에 대하여 border:5px solid red로 심플하게 써도 됨.
